# My Personal Website

A personal site featuring interactive games, dark mode, and cool effects using HTML, CSS, and JavaScript.

## Features

- Interactive Bansuri Flute game with keyboard controls
- Science Quiz game with 5 questions
- Memory Match game with card matching
- Atom Clicker game with a 10 second timer
- Fun facts cards that appear when clicked on
- Toggleable dark mode
- Particles that float around
- Responsive design
- Easter Egg (Konami Code)

## Technologies Used

HTML5, CSS3, JavaScript, Web Audio API

## Running Locally

Local: python3 -m http.server 8000
(where it gets hosted from(the port))

Then open http://localhost:8000

Deploy to GitHub Pages:
1. Push to public GitHub repo
2. Settings > Pages > Deploy from main branch
3. Live in 2-3 minutes
